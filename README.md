# Reflex tesztelő alkalmazás

Egyetemi kurzus keretein belül készített reflex tesztelő játék.

Link a játékhoz: https://reflextestweb.web.app/login

## Technológiák
- Angular 13
- Firestore

## Követelmények
- Regisztráció
- Ranglista
- Billentyűzettel irányítható
